# 📊 Análise de Salários na Área de Dados

## Objetivo
Analisar fatores que influenciam salários na área de dados.

## Perguntas analisadas
- Senioridade impacta o salário?
- Trabalho remoto influencia?
- Empresas maiores pagam mais?
- Existe evolução ao longo dos anos?

## Tec
- Python
- Pandas
- Seaborn
- Matplotlib
- Plotly

## Principais insights
- Senioridade é o fator mais relevante
- Trabalho remoto tem impacto secundário
- Empresas grandes pagam mais
- Salários cresceram ao longo dos anos
